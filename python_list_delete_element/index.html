推导原因如下：
list.remove(i)方法用于删除列表中等于i的第一个元素，这个方法不返回任何值，第一次删除后l变成[2,3,4,5],此时for循环的index增加为1；
第二次遍历时，直接遍历到3，删除3后，index变为2，l变为[3,4,5],然后删除3，依此类推；
