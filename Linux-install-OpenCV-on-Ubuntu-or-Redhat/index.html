我主要在此处遇到了问题，因为我的电脑安装了GPU卡，因此OpenCV会自动编译CUDA相关的代码，可以直接关掉( 在cmake时加上-D WITH_CUDA=OFF )，但我需要编译。因此默认是打开的，因此在make时遇到以下错误：a storage class is not allowed in an explicit specialization
